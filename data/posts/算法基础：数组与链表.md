## 数组

数组是一种线性数据结构，其将相同类型的元素存储在连续的内存空间中。

### 数组常见操作

初始化数组：

```ts
let arr: number[] = new Array(5).fill(0);
let arr1: number[] = [1, 2, 3, 4, 5];
```

访问元素：

```ts
// 返回一个随机索引的数组元素
function randomAccess(nums: number[]): number {
  // 返回一个位于0到nums.length之间的数
  const random_index = Math.floor(Math.random() * nums.length);
  return nums[random_index];
}

```

插入元素：

```ts
function insert(nums: number[], num: number, index: number): void {
  for (let i = nums.length - 1; i > index; i--) {
    nums[i] = nums[i - 1];
  }
  nums[index] = num;
}
```

> 注意是从后往前进行替换，从前往后会将叠加替换，index索引后面的所有元素的值都会等于nums[index]；

删除元素：

```ts
function remove(nums: number[], index: number): void {
    for (let i = index; i < nums.length - 1; i++) {
        nums[i] = nums[i + 1];
    }
}
```

遍历数组：

```ts
function traverse(nums: number[]): void {
    let count = 0;
    // 通过索引遍历数组
    for (let i = 0; i < nums.length; i++) {
        count += nums[i];
    }
    // 直接遍历数组元素
    for (const num of nums) {
        count += num;
    }
}
```

查找元素：

```ts
function find(nums: number[], target: number): number {
    for (let i = 0; i < nums.length; i++) {
        if (nums[i] === target) {
            return i;
        }
    }
    return -1;
}
```

关于数组扩容的说明，由于TS是一种动态类型的语言，可以直接拓展，所以不需要创建一个新的数组然后将旧的数组复制。

### 数组的优缺点

优点： 

- 分配连续的内存空间，空间效率高；
- 支持随机访问，时间复杂度低；
- 访问数组元素在计算机加载时同时还会缓存周围的数据，可以提高后续的执行速度；

缺点：插入和删除需要移动大量的元素，时间复杂度高（但是在TS中首部和尾部的相关操作复杂度很低，只有中间的才比较复杂）；
